<h2 align="center">
  Stripe - UI Clone
</h2>

### Description

This is an UI Clone of Stripe menu, using [Context API](https://reactjs.org/docs/context.html) and [Framer Motion](https://www.framer.com/motion/) library.

### Installation

```bash
$ yarn
```

### Running the app

```bash
$ yarn start
```

### Environment

- Node.js v12.18.3 LTS
